üöÄ Quick Start (Without Firebase)

üîß Firebase Setup (For Full Functionality)

1. Go to Firebase Console
2. Create a new project
3. Enable Firestore Database and Storage
4. Get your Firebase config and replace the placeholder in index.html

üìù How to Use

Creating a Batch:

1. Click "Create Batch" mode
2. Type a word and click "Record Word"
3. Speak clearly, then click "Stop Recording"
4. Click "Play" to verify the recording
5. If satisfied, click "Add to Batch"
6. Repeat for all words, then click "Save Batch"

Practicing:

1. Click "Practice" mode
2. Select a batch from the list
3. Click "Next Word" to hear the first word
4. The word repeats automatically after 3 seconds
5. Continue until all words are practiced
